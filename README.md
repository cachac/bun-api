# bun-api
DEMO: BunJS Rest api

- Rest API ([Hono Framework](https://honojs.dev/))
- JWT auth
- Hot Reload
- CORS
- env variables
- Error Handler
- Routes Folder
- Type: module
- Get POST body request
- No Babel

# Testing
```
bun run dev
or
bun --hot run src/main.js
```
